About MorseCodeX

MorseCodeX is a powerful and user-friendly Morse Code translator that enables seamless conversion between English text and Morse code. Designed for learners, hobbyists, and communication enthusiasts, it brings simplicity and functionality together in a modern web app experience.

Key Features

  • Real-time Translation: Convert instantly between English and Morse code with intuitive feedback.
  • Copy to Clipboard: Easily copy translated Morse or English text with a single click.
  • Translation History: Access and manage your past translations using browser-based localStorage.
  • Clear History: Clean up saved records anytime to start fresh.
  • Audio Playback: Listen to Morse code beeps or have the English translation read aloud.
  • Dark Mode: Toggle between light and dark themes to suit your preference and environment.

This application is built using HTML, CSS, JavaScript, and served via a Python Flask backend.

Our Mission

At MorseCodeX, our mission is to make Morse code learning accessible, intuitive, and enjoyable for everyone. Whether you're exploring Morse as a hobby, studying it for academic purposes, or using it in professional communications, our goal is to provide a reliable and enriching experience.
