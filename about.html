About MorseCodeX

MorseCodeX is a real-time Morse Code Translator built to convert between English text and Morse code seamlessly. It offers a user-friendly interface and is packed with useful features for learners, hobbyists, and enthusiasts of Morse communication.

Key Features:

  • Real-time Translation: Convert instantly between English and Morse code.
  • Copy to Clipboard: One-click copying of translated text or Morse output.
  • Translation History: View and manage past translations right from your browser (stored locally).
  • Clear History: Remove saved translation history when needed.
  • Audio Playback: Hear the Morse code beeps or have the translated text read aloud.
  • Dark Mode: Switch between light and dark themes for comfortable viewing anytime.

This tool was built using HTML, CSS, JavaScript, and Python Flask as the backend for processing translations.
